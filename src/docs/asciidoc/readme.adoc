== Execution
To execute the application run the following command:

.Windows
[subs="attributes"]
----
cmd /C "set JAVA_TOOL_OPTIONS="-Dspring.config.location=C:/path/to/config.yml" java -jar C:/path/to/{myspringapp}.jar"
----

.Unix
[subs="attributes"]
----
JAVA_TOOL_OPTIONS="-Dspring.config.location=/path/to/config.yml" /path/to/{myspringapp}.jar
----
